# P2P Messenger



Мобильное приложение для обмена сообщениями между устройствами Android с использованием технологии Peer-to-Peer соединения без применения центрального сервера передачи данных.



---



## О проекте



P2P Messenger представляет собой мобильный мессенджер, позволяющий пользователям обмениваться текстовыми сообщениями напрямую через локальную сеть.



В отличие от традиционных мессенджеров, использующих клиент-серверную архитектуру, данное приложение использует прямое соединение между устройствами, что позволяет уменьшить задержки передачи сообщений и исключить необходимость использования серверной инфраструктуры.



Проект разработан в учебных целях в рамках дисциплины, связанной с проектированием и разработкой программных систем.



---



## Основные возможности



* создание серверного соединения;

* подключение к удалённому устройству по IP-адресу;

* обмен текстовыми сообщениями в режиме реального времени;

* отображение собственного IP-адреса;

* просмотр встроенной инструкции по использованию приложения;

* завершение соединения пользователем.

## Интерфейс приложения



### Стартовый экран



Здесь пользователь может:



* создать сервер;

* подключиться к удалённому устройству;

* узнать собственный IP-адрес;

* открыть инструкцию.



### Экран инструкций



Содержит краткое руководство по использованию приложения.



### Экран чата



Позволяет пользователям обмениваться сообщениями после установления соединения.
