# Функциональные и нефункциональные требования

## Функциональные требования

Система должна обеспечивать выполнение следующих функций:

-	подключение к выбранному пользователю; 
-	отправка текстовых сообщений; 
-	получение сообщений в реальном времени; 
-	отображение времени отправки сообщений; 
-	отображение статуса соединения; 
-	завершение соединения. 

## Нефункциональные требования

-	время доставки сообщений должно быть минимальным; 
-	интерфейс должен быть интуитивно понятным; 
-	приложение должно корректно работать на устройствах Android; 
-	система должна корректно обрабатывать разрывы соединения; 
-	архитектура приложения должна позволять расширение функционала; 
-	приложение должно обеспечивать стабильную работу при длительном использовании.
